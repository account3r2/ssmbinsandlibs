TO INSTALL:

Copy everything in this folder (excluding this README) into
the root of your AssaultCube directory. The bin_win32
folder should be merged and replace your existing binaries,
and the lua folder should be merged with your existing lua
folder.

NOTE IF INSTALLING (server)^2 Modification:

FOR A CLEAN INSTALL:
DOWNLOAD A FRESH COPY OF ASSAULTCUBE AND INSTALL IT. DELETE
THE bin_win32 FOLDER AND COPY THE ONE FROM HERE INSTEAD.
COPY THE lua FOLDER TO THE ROOT OF YOUR ASSAULTCUBE
DIRECTORY.
